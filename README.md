
# API Login

Esta é uma api com de login com token de autenticação e um CRUD de usuários




## Documentação da API

#### Instalação node_modules

```
  npm install
```

#### Configuração Prisma

```
  npm run generate
```
```
  npm run migrate
```
```
  npm run seed
```

#### Subindo banco postgreSQL com docker-compose

```
  docker-compose up -d
```

#### Iniciando projeto

```
  npm run dev
```
